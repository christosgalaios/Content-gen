Generic Pipeline

Batch content generation from content library + trending topics.

1
Ingest
Scan content folder, extract metadata npm run pipeline:ingest
2
Trends
Scrape TikTok + Claude web search npm run pipeline:trends
3
Plan
AI-plan video content with Claude npm run pipeline:plan
4
Build
Generate Remotion props + validate npm run pipeline:build
5
Render
Batch render videos to MP4 npm run pipeline:render

Event Pipeline

Per-event promotional batches from Meetup scraping.

1
Scrape Events
Extract upcoming events from Meetup npm run pipeline:events:scrape
2
Match Trends
Filter global trends to each event type
3
Plan per Event
Claude generates 2-5 promo videos per event npm run pipeline:events:plan
4
Schedule Posts
Optimal posting times relative to event date (7d, 5d, 3d, 1d before)
5
Build & Render
Generate Remotion props and render npm run pipeline:events

Cloud ↔ Local Workflow

Cloud (GitHub Actions — daily cron)

  • Scrapes Meetup events with Playwright
  • Scrapes TikTok Creative Center trends
  • Commits updated data/pipeline.db to repo
  • No API keys needed — Playwright-only

Local

  • git pull — get latest scraped data
  • npm run pipeline:local — Claude plans content
  • npm run pipeline:build — generate Remotion props
  • npm run pipeline:render — render videos to MP4
